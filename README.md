## 📦 Requeriments
- [NodeJS][nodejs] _v18.14.0_
- [AngularCLI][angularCLI] _v15.2.4_

## 📟 Setup
```bash
$ npm install
$ npm run build
$ npm start
```
> 🛠 Create `.env` file with following content
> ```
> MONGODB_SRV= <string | MongoDB URI>
> JWT_SECRET= <string | unique key define by you>
> ```
